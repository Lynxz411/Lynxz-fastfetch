# Lynxz Fastfetch

Aesthetic boxed Fastfetch config with:
- Nerd Font icons
- OS Age counter
- Kitty image logo
- Clean rounded box layout

## Install
```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/Lynxz411/lynxz-fastfetch/main/install.sh)"
```
## Requirements

- Fastfetch
- Nerd Font
- Kitty terminal (for image logo)
