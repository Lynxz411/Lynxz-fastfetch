# Lynxz Fastfetch

## Preview

![Preview](image.png)


Aesthetic boxed Fastfetch config with:
- Nerd Font icons
- OS Age counter
- Kitty image logo
- Clean rounded box layout

## Install
```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/Lynxz411/Lynxz-fastfetch/main/install.sh)"
```
## Or use this

```bash
git clone https://github.com/Lynxz411/Lynxz-fastfetch.git
cd Lynxz-fastfetch
chmod +x install.sh
./install.sh

## Requirements

- Fastfetch
- Nerd Font
- Kitty terminal (for image logo)
